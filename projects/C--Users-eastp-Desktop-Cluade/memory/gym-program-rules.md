도요의 헬스장 최소 원판은 2.5kg(페어)이므로 **바벨 무게는 반드시 5kg 단위로만 처방**한다 (72.5kg 같은 2.5 단위 불가). 머신·덤벨은 기구 단위 따름.

운동 프로그램은 구글드라이브 `운동기록.xlsx`(2023) 4주 파동식의 **x0.8 재현**: W1 7회@6.5 → W2 5회@7 → W3 3회@8 → W4 싱글@9. 선형 증량이 아니라 주차마다 회수↓·무게↑.

**Why:** 260720 도요가 직접 교정 — 2.5 단위 처방(72.5/82.5)과 W1 반복 처방(또 7회)을 둘 다 지적함.

**How to apply:** 운동 루틴을 짤 때 vault `30_Areas\건강_운동\운동기록.md`에서 지난 세션의 주차(회수)를 확인하고 다음 파동 단계로, 무게는 5kg 단위 반올림으로 처방한다.

추가 규칙 (260720): **편측 운동(런지·싱글암 등)은 왼쪽(왼발/왼손) 먼저** — 처방·기록 순서 모두. 루틴은 운동 전날 밤~당일 아침에 확정본으로 전달하고, 운동 중에 계획을 바꾸지 않는다.
