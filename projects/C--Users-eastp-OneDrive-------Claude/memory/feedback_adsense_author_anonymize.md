애드센스 블로그([[project-adsense-blog]]) 글의 저자 소개·경험 서술은 실제 회사명(SK에코플랜트)이나 식별 가능한 프로젝트(반도체 FAB / SK하이닉스)를 쓰지 않고, **"설계사무소·건설사에서의 건축 실무 경험(설계 조율·인허가)"**으로 일반화한다.

**Why:** 개인 수익 블로그를 고용주와 연결시키지 않기 위함(신원 보호). 도요 지시 2026-07-05.

**How to apply:** 모든 애드센스 원고·글템플릿·슬라이드(대표 이미지 footer 포함)에 적용. "앞으로도 쭉" = 표준 규칙. 글템플릿.md 저자 소개 항목에도 반영됨.
