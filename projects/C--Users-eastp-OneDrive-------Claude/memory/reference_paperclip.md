에이전트 비용추적·승인게이트·현황 모니터용 **Paperclip** 설치됨. 경로 `C:\paperclip`, 로컬 `http://127.0.0.1:3100`, 임베디드 Postgres DB, 회사명(Workspace) "DM Studio". 서버 실행: `pnpm -C C:\paperclip dev`.

현재 말숙이팀과 **미연동**(설치만). 연동 시 REST API + 웹소켓(에이전트 API 키)으로 대시보드 구성 가능. 사용법: vault `40_Resources\AI공부\260701_Paperclip_사용설명서.md`.
