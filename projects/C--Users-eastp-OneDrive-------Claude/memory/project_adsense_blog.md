  1. 해자 필러: 건축 법규·인허가 실무 해설 + 국민신문고 사례 (도요 인허가 경험=E-E-A-T). 승인용=명확·사실형(갈래1), 분쟁성 사례(갈래2)는 승인 후. YMYL이라 방어장치 3종(출처+개정일/법률자문아님 디스클레이머/조례차이 경고) 필수. 한국 전용(영어 이식 X). ※2026-07-05 자동화→법규해설로 방향 변경

  3. 트렌드/영어확장 필러: AI 도구·생산성 (검색 폭발, 영어 이식 쉬움)
