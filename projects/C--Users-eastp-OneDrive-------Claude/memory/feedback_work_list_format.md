업무목록(회사업무 리스트) 관련 2가지 규칙:

1. **메모는 조사를 최대한 빼고 개조식으로 작성한다.** 예: "간섭이 발생하여" → "간섭 발생", "확인이 필요해" → "확인 필요".
2. **"업무 확인해줘" 류 요청에는 표(table) 형식으로 답한다.** 채팅 텍스트로 줄글 나열하지 않는다.

**Why:** 도요님이 2026-07-13에 명시적으로 요청함. 업무목록.md는 SK하이닉스 본업 업무 대시보드 스킬([[project_work_priorities]])의 원본 데이터.

**How to apply:** `/업무-관리` 스킬로 업무목록.md를 갱신할 때마다 이 문체 규칙 적용. 업무 현황 질문에는 ID·제목·대응처·담당자·상태·최근 로그 요약을 표로 정리해서 답한다.
