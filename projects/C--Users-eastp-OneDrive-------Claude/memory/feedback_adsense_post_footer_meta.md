애드센스 블로그([[project-adsense-blog]]) 원고(`원고.md`) 작성 시, **맨 아래에 항상** 아래 블록을 포함한다.

```
## 발행 메타 (블로그스팟 복붙용)
**[라벨(태그)]**: 키워드 4~6개, 큰 카테고리 우선
**[검색 설명]**: 메타 디스크립션과 동일 문구(155자 내)
```

**Why:** 도요가 매번 물어보지 않고 블로그스팟에 바로 복붙할 수 있게. 발행 단계에서 "라벨 뭐 넣지, 검색설명 뭐지" 왔다갔다하는 마찰 제거.

**How to apply:** [[feedback-adsense-slide-standards]]·[[feedback-adsense-author-anonymize]]와 같은 층위의 표준. `10_기획/글-템플릿.md`의 "8. 발행 메타" 항목에도 반영됨 — 새 글 작성 시 템플릿 그대로 따르면 자동 충족.
