# 블로그-작성 스킬

## 사용법
```
/블로그-작성 <폴더경로 또는 파일경로들> [주제]   ← 모드 1 (사진 기반)
/블로그-작성 <주제 텍스트>                        ← 모드 2 (주제 자동)
```
예시:
```
# 모드 1: 사진 기반 체험기
/블로그-작성 C:\Users\eastp\Pictures\더현대\
/블로그-작성 C:\Users\eastp\Pictures\IMG_001.jpg 주말 나들이

# 모드 2: 사진 없이 주제만 → 자동 정보글
/블로그-작성 2026 아빠 육아휴직 제도 총정리
/블로그-작성 신생아 분유 고르는 법
```

## 말숙이 동작: 파싱 후 즉시 위임

말숙이는 아래 2단계만 수행한다. 실제 작업은 **문이**가 전담한다.

### 1단계: 입력 파싱 + 모드 판별

`$ARGUMENTS`를 보고 이미지 경로 유무로 모드를 판별한다.
- 폴더 경로/파일 경로(.jpg/.png 등)가 있으면 → **모드 1 (사진 기반)**
  - 폴더 경로 → 이미지 파일 목록 / 파일 경로 → 해당 파일들 / 나머지 → 주제
- 이미지 경로 없이 텍스트만 있으면 → **모드 2 (주제 자동)**
  - 전체 텍스트를 주제로 간주

### 2단계: 문이에게 즉시 위임 (백그라운드)

파싱 결과 + 판별한 모드를 문이(Agent)에게 **`run_in_background: true`로** 전달한다.
백그라운드 실행이므로 말숙이는 즉시 대기 해제되어 도요님의 다른 지시를 받을 수 있다.
모드 2에서는 문이가 캐순이·물색이·손질이를 직접 부린다 (말숙이는 관여 안 함).

```
문이에게 전달:
- 모드 1: 이미지 경로 목록/폴더 + 주제(있으면)
- 모드 2: 주제 텍스트
- run_in_background: true
```

문이가 완료 보고를 보내면, 말숙이가 도요에게 결과를 전달한다:
- 저장된 파일 경로
- (모드 1) 이미지 복사 경로 / (모드 2) 글 유형·출처 수·검수 통과 여부

말숙이는 조사, 글쓰기, 검수, 저장을 직접 수행하지 않는다.
