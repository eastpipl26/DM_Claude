# 부동산-분석 스킬

## 사용법
```
/부동산-분석 <단지명 또는 지역>
/부동산-분석 송도 SK뷰 8공구
/부동산-분석 송도 마스터뷰 21단지, 22단지, 글로벌파크베르디움 비교
```

---

## 말숙이 동작: 파싱 후 즉시 위임

말숙이는 아래 2단계만 수행한다. 실제 작업은 탐이(크롤링) → 감이(분석)가 전담한다.

### 1단계: 입력 파싱

입력에서 단지명·지역·비교 여부를 파싱한다.
- 단지명 1개 → 단일 분석
- 단지명 2개 이상 → 비교 분석

### 2단계: 탐이 → 감이 순서로 위임

**탐이에게 먼저 위임:**
- 대상 단지별 네이버부동산 매물 크롤링
- 국토부 실거래가 API 조회 (최근 2년)
- 호갱노노 시세 크롤링
- 단지 입지 관련 뉴스 (호재·교통·개발) 크롤링

**탐이 완료 후 감이에게 위임:**
- 수집 데이터 기반 과거·현재·미래 시세 분석
- 입지 종합 평가 (교통·교육·호재·인프라)
- 단지별 비교표 + 점수 산출
- 최종 투자 제안 (TOP 순위 + 관망/매수 판단)

말숙이는 감이 완료 보고 수신 후 도요님께 결과 전달.
