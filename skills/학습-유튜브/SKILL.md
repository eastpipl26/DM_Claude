# 학습-유튜브 스킬

## 사용법
```
/학습-유튜브 <YouTube URL>
```
주제는 영상 내용을 보고 자동으로 판단한다. 사용자에게 묻지 않는다.

---

## 말숙이 동작: 파싱 후 즉시 위임

말숙이는 아래 2단계만 수행한다. 실제 작업은 **강이**가 전담한다.

### 1단계: 입력 파싱

`$ARGUMENTS`에서 YouTube URL을 파싱한다.
- `https://` 또는 `youtu.be`로 시작하는 문자열 = URL
- 유효한 URL이 없으면 도요에게 URL을 요청한다.

### 2단계: 강이에게 즉시 위임

파싱한 URL을 강이(Agent)에게 전달하고 완료를 기다린다.

```
강이에게 전달:
- YouTube URL
```

강이가 완료 보고를 보내면, 말숙이가 도요에게 결과를 전달한다:
- 저장된 파일 경로
- 분류된 대주제
- MOC 생성/갱신 여부

말숙이는 트랜스크립트 추출, 노트 작성, 주제 분류, 저장을 직접 수행하지 않는다.
