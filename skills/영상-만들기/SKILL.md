# 영상-만들기 스킬

Higgsfield 기반 설명영상(explainer) 제작 워크플로.
힉스필드·애니메이션 스튜디오 세션에서 반복된 작업 패턴을 표준화한 것.

## 사용법
```
/영상-만들기 <주제 또는 대본 파일 경로>
```
예시:
```
/영상-만들기 아기 개월수별 놀이법
/영상-만들기 {vault}\30_Areas\콘텐츠\260712_대본.md
```

## 워크플로

1. **대본 준비**: 파일 경로면 읽고, 주제면 블록(장면) 단위 대본 작성 → 도요 확인.
2. **워크플로 로드**: `mcp__higgsfield__get_workflow_instructions` 먼저 호출(카탈로그 확인 후 매칭 워크플로 로드). 임의로 절차를 추측하지 않는다.
3. **크레딧 게이트 (필수)**: 크레딧이 소모되는 생성 호출 전에 예상 소모량(클립 수 × 단가)을 도요에게 알리고 승인받는다. 잔액 부족·충전 필요 상황도 즉시 보고.
4. **생성 실행**: 보이스오버 → 키프레임/클립 → 최종 합성(자막 포함) 순.
5. **오탐 처리**: nsfw 등 콘텐츠 필터 오탐으로 실패한 블록은 2회까지 프롬프트를 바꿔 재시도, 계속 실패하면 해당 블록 제외하고 어떤 블록이 왜 빠졌는지 보고.
6. **완료 보고**: 최종 MP4 링크 + 포함/제외 블록 수 + 총 소모 크레딧 + 잔여 크레딧.

## 하지 말 것

- 승인 없이 크레딧 소모 생성 호출 금지.
- 시스템이 추천하는 프리셋이 컨셉과 안 맞으면 무시하고 원래 기획 유지 (선례: "IN THE DARK" 오추천).
- 로컬 무료 렌더링(hyperframes)이 더 적합한 짧은 모션그래픽·자막 작업이면 그쪽 스킬을 먼저 제안.
