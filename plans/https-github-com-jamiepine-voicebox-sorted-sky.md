# 플랜: 문이 확장 — 사진 없는 주제 기반 자동 블로그 + 서브에이전트 팀

## Context

도요 요청: 문이가 "내가 준 사진"에만 의존하지 말고, **사진 없이 주제만으로 AI가 자동으로 글을 쓰는 모드**를 갖췄으면 함. 리서치는 자동 웹검색.

조사 결론 (2026-06):
- **독자는 사진 너머 정보를 원함**: 위치·영업시간·주차·가격·웨이팅 등 실용정보가 체류시간·검색유입을 좌우. → 사진만으로는 한계.
- **멀티에이전트 글쓰기 표준 = 리서처 → 키워드/전략 → 작가 → 에디터(사실검증)**. 분업이 환각을 줄이고 품질을 올림. 핵심: "사실 검증을 글쓰기 전에 끝낸다".

### 중요한 전제 (저품질·신뢰 회피)
- 사진 없는 자동 글은 **체험기(다녀온 후기) 금지** — 안 해본 경험을 지어내면 네이버 저품질 분류 + 신뢰 붕괴.
- 대신 **정보·정리형 글**로 한정하면 완전 자동화 가능하고 진정성도 유지.
- 글 성격을 정직하게: "정보 정리" 글임을 톤으로 드러내고, 가능하면 도요 관점(건축PM·AI 전환)을 얹어 차별화.

### 자동화 가능한 글 유형 (사진 불필요)
1. **정보 큐레이션** — 예: "2026 육아휴직·아빠 출산휴가 제도 총정리"
2. **비교·랭킹** — 예: "신생아 분유 5종 비교", "용인 가볼만한 곳 정리"
3. **하우투·가이드** — 예: "건설 현장에서 엑셀 자동화 시작하는 법"
4. **트렌드·뉴스 해설** — 시사·기술 이슈 쉽게 풀기
5. **도요 전문 칼럼** — 건축 PM × AI 전환 (개인 브랜딩, 가장 차별화)

## 구현 계획

### A. 문이 = 듀얼 모드로 재정의

| 모드 | 입력 | 동작 |
|------|------|------|
| **모드 1: 사진 기반** (기존) | 사진 폴더(+주제) | 지금 그대로. 체험기·일상·에세이 |
| **모드 2: 주제 기반 자동** (신규) | 주제 텍스트만 | 서브에이전트 팀이 조사·검증, 문이가 정보형 글 작성 |

`/블로그-작성` 입력에 이미지 경로가 없고 주제 텍스트만 있으면 → 모드 2 자동 진입.

### B. 신규 서브에이전트 3종 (확정)

분업 구조로 환각을 줄이고 품질을 올린다. 이름은 역할-비틀기 룰(선우=성우), 두 글자 + 이.

1. **캐순이** (캐다·캐묻다) — 리서치 에이전트
   - 주제로 **자동 웹검색** → 사실·통계·최신정보·실용정보 수집
   - 출처 신뢰도 티어 표시([T1/T2/T3]) — 새벽이 방식 재사용
   - 환각 방지: 출처 없는 내용은 자료팩에 넣지 않음
   - 산출물: 구조화된 "자료팩"(사실 + 출처 URL) → 문이에게 전달

2. **물색이** (物色) — 키워드/SEO 에이전트
   - 네이버 연관검색어·인기 키워드·상위 노출 글 구조 조사
   - 제목(질문형 옵션)·소제목·태그 후보 제안 (C-Rank/DIA)
   - 산출물: 키워드 세트 + 추천 제목/구조

3. **손질이** (손질하다) — 에디터·검수 에이전트
   - **발행 전 게이트**: 문이 초안을 자료팩과 대조해 사실 검증 (미출처 주장 플래그)
   - 맞춤법·가독성·저품질 4종 검열(상업성 키워드·중복표현·중복이미지·과장)
   - 통과해야 저장 진행 ("검증 green 전엔 미완료")

### C. 모드 2 파이프라인 (말숙이 위임 흐름)

```
도요: /블로그-작성 "2026 아빠 육아휴직 제도 정리"
  ↓ 말숙이: 이미지 없음 감지 → 모드 2, 문이에게 위임
문이가 오케스트레이터로서:
  1. 캐순이 호출 → 자료팩(사실+출처)
  2. 물색이 호출 → 키워드·제목 구조
  3. 문이 직접 작성 (자료팩 기반, 도요 관점 가미, 3줄요약·소제목)
  4. 손질이 호출 → 사실대조·검열 (실패 시 1회 보완)
  5. 옥순이에게 저장 위임 + 시리즈 연결
  6. 말숙이에게 완료 보고
```

### D. 이미지 처리 전략 (저작권 안전)

사진 없는 글의 시각 요소:
- 저작권 있는 웹이미지 **무단 사용 금지**
- 본문에 `📷 [여기에 직접 사진/캡처 추가]` 자리표시 + 무엇을 넣으면 좋을지 제안
- 정보형 글은 **표·체크리스트·요약박스·비교표**로 시각 보강 (이미지 의존도↓)
- 공식 출처는 링크로 안내 (지도·기관 페이지 등)

## 검증 방법

4. 에디터 게이트에서 저품질 4종·맞춤법 검열 통과 확인
5. 기존 모드 1(사진 기반)도 그대로 동작하는지 회귀 확인
```
