# NoRender 브랜드 아이덴티티 가이드라인 v2 — 최종 계획

## Context

지난번(2026-07-10) 만든 `260710_NoRender_BrandSheet_v1.png`는 로고+컬러+명함목업을 한 판에 욱여넣은 "무드보드"였다. 실제 요청은 섹션이 나뉜 "가이드라인"이었고, 진행 중 다음이 확정/변경됐다:

- 회색 단색 팔레트는 브랜드 초창기 컨셉(초현실적·황혼빛 무드)을 못 살려 폐기, 아이덴티티 컬러로 교체
- 로고 도면 관습(치수선·인출선·타이틀블록·모눈그리드)을 그래픽 언어로 채택 — 건축 조형물은 넣지 않음
- 서체 확정: Posterama 2001 (NORENDER=Light, "~STUDIO"=Thin)
- 페이지 01의 "오용 사례" 섹션은 불필요 — 대신 SNS 플랫폼별(인스타/유튜브 등) 로고 활용 사이즈 가이드로 대체

## 확정 사양

### 컬러 (제안, 그림 생성 전 최종 확인)
| 역할 | 컬러 |
|---|---|
| Primary (잉크) | Deep Charcoal `#1C1C1E` |
| Accent (정체성) | Midnight Blue `#141B2E` |
| Neutral | Warm Greige `#C9C4B8` |
| Base | Matte White `#F7F6F3` |

### 서체
- NORENDER 워드마크: Posterama 2001 Light
- "~STUDIO" 태그라인: Posterama 2001 Thin

### 그래픽 언어 (전 페이지 공통)
- 치수선·인출선(dimension/leader line)으로 규칙 표기
- 페이지 상단/하단에 타이틀블록: "NORENDER IDENTITY GUIDE / SHEET 0X of 03 / SCALE N.T.S. / 2026"
- 옅은 배경 모눈그리드
- 모서리 정렬마크(크롭마크 스타일)
- 건축 조형물·건물 실루엣 절대 금지

## 페이지 구성 (3장, 각각 별도 이미지 생성)

### 01. LOGO
- 풀 로고타입(NORENDER + 태그라인) + 모노그램(NR)
- 클리어스페이스 규칙 (치수선으로 표기)
- 최소 사이즈: 인쇄(mm) / 화면(px) — 대/중/소 3단계
- **NEW — SNS 플랫폼 사이즈 가이드** (오용 사례 대체):
  - Instagram 프로필: 320×320px (원형 세이프존)
  - Instagram 정사각 포스트: 1080×1080px
  - YouTube 채널 아이콘: 800×800px
  - YouTube 채널 아트/배너: 2560×1440px (세이프존 1546×423px)
  - 카카오톡/네이버 프로필: 1:1 비율
  - 표기는 인출선+치수 스타일로 통일 (다른 규칙들과 그래픽 언어 일관성 유지)
- 오용 사례 섹션 없음

### 02. COLOR & TYPE
- 컬러 4종 스와치 (위 표, HEX/RGB/CMYK 병기)
- 타이포 스펙: Posterama 2001 Light(워드마크)/Thin(태그라인) 알파벳 전체 세트 + 숫자
- 본문용 보조 서체 샘플

### 03. APPLICATION
- 명함 목업 (모노그램+워드마크)
- SNS 프로필 적용 목업 (실사용 예시 1컷)
- 파비콘/워터마크 소스
- **워터마크 배치 예시**: 렌더링 이미지 위 NR 모노그램 배치 — 위치(우하단 기본), 여백(가장자리로부터 치수 표기), 불투명도(%) 규정을 인출선 스타일로 표기
