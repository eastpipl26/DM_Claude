## 네이밍 기준

신규 직원 이름을 지을 때 반드시 아래 원칙을 따른다.

1. **실제 한국 이름이어야 한다** — 사람 이름처럼 들릴 것. 외래어 음차 금지 (예: 파이이, 인세이 ❌)
2. **역할을 한자·순우리말로 연상 후 비틀기** — 역할 → 한자어 → 이름으로 가져오기
3. **직관적 연결 금지** — 이름만 보고 역할을 바로 알면 안 됨. 알고 나면 "아" 하는 느낌
4. **2~3음절 + 이(이름 어미)** 권장 — 강이, 문이, 재이, 옥순이 등

**이름 짓기 프로세스:**
```
역할의 핵심 동작 → 한자어로 변환 → 그 음으로 한국 이름 조합 → 실제 이름처럼 들리는지 확인
예) 크롤링(탐색探索) → 탐(探)에서 '태민' → 태민이
예) 음성(聲) → 선(仙/宣)에서 '선우' → 선우
```

## 평가 기준 (공통)

| 기준 | 가중치 | 측정 방법 |
|------|--------|-----------|
| 결과 정확도 | 40% | 기대 출력과 일치 여부 |
| 안정성 | 30% | 3회 실행 중 오류 횟수 |
| 속도 | 20% | 평균 응답 시간 |
| 설치·설정 난이도 | 10% | 설치 단계 수, 외부 의존성 |

**채용 기준:**
- 총점 70점 이상 (상=3점, 중=2점, 하=1점으로 환산) → 정식 채용
- 70점 미만 → 대기발령 (사유 명시)
- 경쟁 채용 시: 총점 높은 쪽 우선, 동점이면 정확도 기준
