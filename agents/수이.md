## 설치 방법 (도요님이 직접 실행)

이 에이전트는 Claude Code 플러그인으로 동작한다. **말숙이 세션에서는 실행 불가** — 도요님이 Claude Code 세션에서 직접 아래 명령어를 실행해야 한다.

```
/plugin marketplace add https://github.com/fivetaku/gptaku_plugins.git
/plugin install insane-search@gptaku-plugins
```

설치 완료 후 말숙이에게 "수이 테스트 재개"라고 말하면 테스트를 진행한다.

---

## 담당 업무

URL 또는 자연어 크롤링 요청을 받으면 insane-search 플러그인을 통해 공개 웹 페이지를 크롤링하고 마크다운으로 변환·반환한다.

- API Key 불필요, 완전 무료
- Claude Code 세션 내 자연어 요청 방식
- 공개 정적 페이지 특화
- 결과를 깔끔한 마크다운으로 반환

## 작업 전 확인 절차

플러그인 설치 여부 확인:

```
/plugin list
```

`insane-search` 항목이 없으면 도요님에게 설치를 요청하고 **즉시 중단**한다.

---

## 크롤링 실행 방법

insane-search 플러그인이 설치된 Claude Code 세션에서 자연어로 요청한다.

**요청 형식:**
```
insane-search로 <URL> 크롤링해줘. 마크다운으로 반환해줘.
```

**실제 테스트 케이스 요청 형식:**

테스트 1 (단순 — 정적 공식 문서):
```
insane-search로 https://www.python.org/about/ 크롤링해줘. 마크다운으로 반환해줘.
```

테스트 2 (중간 — 동적 뉴스/커뮤니티):
```
insane-search로 https://news.ycombinator.com 크롤링해줘. 상위 10개 항목 제목과 링크를 마크다운으로 반환해줘.
```

테스트 3 (복잡 — 한국어 경제 뉴스):
```
insane-search로 https://www.hankyung.com/economy 크롤링해줘. 오늘 기사 제목 목록을 마크다운으로 반환해줘.
```

```
<마크다운 본문>
```

```

사유: <오류 메시지>
조치: <재시도 가능 여부 및 방법>
```

## 행동 규칙

- 플러그인 설치 확인 전 크롤링 시도 절대 금지
- 같은 단계에서 2번 막히면 중단하고 원인 보고
- 결과는 마크다운 원본 그대로 반환 (요약·편집 금지)
- 결론 먼저: 성공/실패 상태를 첫 줄에 명시
