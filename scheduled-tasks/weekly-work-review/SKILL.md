## 수행 절차
1. items\*.md 전체를 읽고 ✅완료 제외한 미결 항목을 분류한다:
   a. **F/U 필요**: 상태 ⏳대기이고 마감일(=회신기한)이 오늘 이전 → 재촉 필요 목록
   b. **재분류 필요**: 상태 🟡진행이고 마감일이 오늘 이전 → 지연/대기/완료 확인 필요 목록
   c. **날짜 미정 확인건**: 메모 로그에 "확인 필요"가 있는데 확인 날짜가 안 잡힌 항목
2. 오늘 데일리노트의 "## 🎯 오늘 할 일" 섹션에 위 목록을 체크리스트로 추가한다. 형식: `- [ ] [ID](../../../20_Projects/SK하이닉스_용인/업무/items/파일명.md) 요약 (F/U|재분류|확인)`. 이미 있는 항목은 중복 추가하지 않는다.
3. 항목 파일 자체는 수정하지 않는다 (상태 변경은 도요님 확인 후 메인 세션에서).
4. 완료 후 요약 보고: F/U 몇 건, 재분류 몇 건, 확인 몇 건 + 데일리노트 절대경로.

주의: Notion은 건드리지 않는다 (요청 시 일괄 동기화 원칙).
