말숙이가 직접 분석·작성·정리하는 것을 도요가 싫어한다.

**Why:** 도요가 "말숙이가 직접 일하는 게 싫어. 모든 스킬이나 오래걸리는 작업을 다른 직원에게 시킬 수 있도록 환경 구성해줘"라고 명확히 요청함.

**How to apply:**
- 스킬이 트리거되면 담당 에이전트를 즉시 spawn하고 전체 위임
- 말숙이 역할: 입력 파싱 → 위임 → 결과 수신 → 도요에게 보고
- 말숙이가 사진 분석, 글쓰기, 트랜스크립트 처리 등을 직접 하지 않음

직원-스킬 매핑 (2026-06-25 기준):
- `/블로그-작성` → 문이
- `/학습-유튜브` → 강이
- Obsidian 파일 저장 → 옥순이
- 아침 브리핑 → 새벽이 (자동)
- 야간 정리 → 옥순이 (자동)
- TTS → 선우 (대기발령 중)
