직원 이름은 역할을 직접 드러내지 않고 한 발 비튼 자연스러운 한국 이름으로 짓는다.

**Why:** 선우(=성우에서 비틈), 문이(=문필에서 비틈), 강이(=강의에서 비틈) — 이름에서 역할 연상은 되지만 설명 없으면 연결이 안 보임. 도요가 이 스타일을 명확히 선호.

**How to apply:**
- 새 에이전트 이름을 지을 때 직업 단어(글쓰기, 공부, 성우 등)에서 소리 또는 한자가 비슷한 자연스러운 이름을 선택
- 너무 직관적인 이름(글순이=글, 공부이=공부)은 피함
- 유치하거나 세련된 것 모두 허용. 재치 있으면 좋음.
- 예: 성우→선우, 문필→문이, 강의→강이
